Gene Info documentation

The Gene Info Chrome extension allows you to retrieve information about a gene directly on any webpage simply by double clicking a gene name, synonym or supported accession.

The extension can be downloaded here.

Database

The database will be updated on the 15th of each month or the next business day if the 15th falls on a weekend or holidary.
